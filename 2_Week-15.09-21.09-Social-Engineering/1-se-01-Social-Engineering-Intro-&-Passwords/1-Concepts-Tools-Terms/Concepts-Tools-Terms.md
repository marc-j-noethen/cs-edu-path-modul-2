# Social Engineering Intro and Passwords

## Summary Based on the 80/20 Principle

### 1. What this topic is about
This topic introduces social engineering, common manipulation tactics, and why passwords are still one of the most valuable attacker targets.

### 2. Key ideas
- Social engineering attacks people, trust, habits, and urgency rather than only technology.
- Common levers include authority, urgency, fear, curiosity, and helpfulness.
- Password theft is a major outcome because a single reused or weak password can unlock multiple systems.
- Strong password hygiene depends on long unique passwords, password managers, and multi-factor authentication.

### 3. Tools and practical relevance
- This topic is mainly conceptual rather than tool-driven.
- Password managers are a practical defensive control because they support unique credentials at scale.

### 4. Key takeaway
People are part of the attack surface. Security awareness and good credential hygiene are core defensive controls.

## Table 2: Technical Terms

| Term | Meaning |
|---|---|
| Social engineering | Psychological manipulation to influence user behavior |
| Phishing | Fraudulent messaging to steal data or trigger actions |
| Pretexting | Creating a believable story to gain trust or access |
| Password reuse | Using the same password across multiple services |
| Hashing | Transforming data into a one-way digest for storage or checking |

## Table 3: Key Vocabulary

| Term | Meaning |
|---|---|
| trust | confidence an attacker tries to exploit |
| urgency | pressure used to reduce careful thinking |
| credential | login data such as username and password |
| reuse | using the same secret again in another place |
| breach | unauthorized disclosure or compromise |
